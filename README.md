# Ascii_Game_Engine--C-plus-plus
Built a modular ASCII game engine in C++ using ncurses, emphasizing clean design and ease of use. Implemented continuous collision detection for smooth gameplay and applied object-oriented patterns like Strategy to enable flexible, dynamic entity behavior.

## Example Games

To demonstrate and validate the functionality of the game engine, two sample games were developed using the engine: **Space Invaders** and **Flappy Bird**.
